TodoMVC Challenge

This challenge consists of trying to use Assetgraph to build every example application implemented in TasteJS's TodoMVC project.

I will log my progress with building each of the apps, showing you build success or failure, what steaps I had to take to make the build work and what bugs I found and had to fix.

My end goal with taking this challenge upon myself is to prove the worth of assetgraph as a build system stable enough to be a plug in replacement for the current Grunt build chain that is auto generated by most Yeoman generators.
